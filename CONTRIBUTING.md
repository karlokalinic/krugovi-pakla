# Contributing

Contributions are welcome for design, narrative, performance, accessibility, and technical refinement.

## Development workflow

1. Install dependencies with `npm ci`.
2. Start the app locally with `npm run dev`.
3. Run `npm run check` before submitting changes.

## Guidelines

- Keep the experience accessible and respectful of the source material.
- Prefer clear, maintainable TypeScript over clever shortcuts.
- Avoid introducing unnecessary dependencies.
- Document significant changes in the changelog.
